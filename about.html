Texplorer

Texplorer is a mobile friendly map that allows a user to see Texas State Historical Commission historic and architectural markers within a 16 km radius of their location, and a timeline of key dates from state history. The tool is intended for use by heritage tourists, students, and others wishing to see nearby historical markers in a context which combines goelocation and chronology.
